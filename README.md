# robot-foobar-alma

## Goal
Manage ressources by using Robot to mine what you need to create foobar
The goal is to have 20 robots
The rest is ingame
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
